# BMC Medical Informatics and Decision Making LaTeX Package

This folder contains the manuscript source prepared for BMC Medical Informatics
and Decision Making using the Springer Nature `sn-jnl` LaTeX template.

Compile target:
- `main.tex`

Included manuscript assets:
- `references.bib`
- `sn-jnl.cls`
- `sn-vancouver-num.bst`
- `figures/`
- `source_data/`
- `Additional_file_1_source_data_and_code.zip`
- `Additional_file_2_reporting_checklist_mapping.md`

Submission notes:
- Upload this folder or `bmc_midd_latex_upload.zip` as the LaTeX source package.
- Upload `Additional_file_1_source_data_and_code.zip` as Additional file 1.
- Upload `Additional_file_2_reporting_checklist_mapping.md` as Additional file 2 if the journal portal accepts checklist-support files.
- Raw individual-level public-use records are not redistributed.
- The corresponding author email is included in `main.tex`; non-corresponding
  author emails are not listed in the manuscript source.
